# K-Fuel Fair Price

국제 석유제품 가격, 국내 주유소 가격, 정책 변수, 공간 격자 데이터를 활용해 국내 유가의 적정 가격을 추정하고 시각화하는 프로젝트입니다.

현재 레포는 두 축으로 운영합니다.

- `analysis-modeling/`: 데이터 분석, 전처리, 시차 분석, 정책 적용, 예측 모델 제작 과정을 단계별로 정리합니다.
- 레포 최상단: 추후 GitHub Pages와 GitHub Actions 기반 자동 추론 웹사이트 코드를 둡니다.

원본 Colab 노트북은 그대로 보관하지 않고, 단계별 폴더의 `code.ipynb`로 나누어 정리합니다.
