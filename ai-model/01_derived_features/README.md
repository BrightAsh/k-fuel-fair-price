# 01. Derived Features

AI 01은 최종 격자 패널을 만들기 전, 원천 주유소/시설/공시지가 데이터를 격자화 가능한 파생 데이터로 정리하는 단계입니다. 직접 학습 target을 만들지는 않고, AI 02가 날짜 x 500m 격자 패널을 만들 수 있도록 입력 재료를 표준화합니다.

## 단계 목적

| 데이터 | 목적 |
|---|---|
| 주유소 위치 이력 | 특정 날짜에 주유소가 어느 500m 격자에 속하는지 결정 |
| 주유소 속성 이력 | 브랜드, 셀프 여부 등 가격 차이를 설명할 수 있는 속성 제공 |
| 시설 좌표 | 저유소, 대리점, 공장과의 거리 기반 영향권 feature 생성 |
| 공시지가 snapshot | 토지 가격 수준을 저빈도 격자 feature로 제공 |
| 전국 land grid | 주유소가 없는 지역까지 포함하는 500m 분석 단위 정의 |

## 코드 구성

| 파일 | 역할 |
|---|---|
| `01_derived_features.ipynb` | 파생 데이터 생성 노트북 |
| `01_derived_features.py` | 같은 로직을 담은 Python 코드 |

코드는 입력 준비 확인, 전국 일별 feature 생성, 주유소 위치/속성 이력 생성, 시설 좌표 보강, 전국 500m land grid 생성, 공시지가 격자화 순서로 구성됩니다.

## 주요 로직

### 1. 전국 일별 feature

원유, 국제제품가격, 환율, 전국 평균 가격, 브랜드 가격, 세금, 정유사 가격 중 날짜 컬럼을 안정적으로 해석할 수 있는 데이터를 일별 테이블로 표준화합니다. 최신 산출물 `national_daily_features.csv`는 6,736행, 21열이며 기간은 2008-01-01 ~ 2026-06-10입니다.

### 2. 주유소 위치/속성 이력

지역별 주유소 metadata와 가격 wide table을 결합해 주유소별 위치 이벤트와 속성 이벤트를 만듭니다. 같은 `station_id`가 여러 지역에서 중복 등장해도 최종 포인트에는 하나의 대표 좌표만 남깁니다.

| 산출물 | 내용 |
|---|---|
| `station_location_history.csv` | `station_id`, 위치, 적용일 이력 |
| `station_attribute_history.csv` | 브랜드, 셀프 여부 등 속성 이력 |
| `station_latest_profile.csv` | 최신 주유소 프로필. 좌표 결측 행 포함 |
| `station_points.csv` | 유효 좌표만 남긴 주유소 포인트 |

### 3. 시설 좌표

시설 데이터는 유형을 `storage`, `agency`, `factory` 중심으로 표준화합니다. 좌표가 있는 시설은 AI 02에서 거리 감쇠 영향권 feature로 변환됩니다.

| 산출물 | 내용 |
|---|---|
| `facility_points.csv` | 전체 시설 좌표 결과. 좌표 결측 포함 |
| `facility_location_data_final.csv` | 유효 좌표만 남긴 시설 포인트 |

### 4. 전국 500m land grid

전국 land geometry와 주유소/시설 주변 patch grid를 결합해 500m 격자틀을 만듭니다. 섬 지역 누락을 줄이기 위해 guard island를 추가합니다. 이 격자는 주유소가 없는 지역도 포함하므로, 이후 웹 지도와 데이터 coverage 분석에도 사용됩니다.

### 5. 공시지가 격자

공시지가 원자료를 500m 격자 기준 snapshot 컬럼으로 정리합니다. 최신 snapshot 컬럼은 `20160921`, `20171121`, `20181129`, `20190820`, `20200813`, `20211123`, `20220805`, `20231221`, `20240802`입니다.

## 좌표 품질

| 구분 | 전체 행 | lon/lat 결측 | 숫자 좌표 범위 이탈 | 유효 좌표 |
|---|---:|---:|---:|---:|
| 주유소 | 28,828 | 415 | 0 | 28,413 |
| 시설 | 797 | 57 | 0 | 740 |

범위를 벗어난 숫자 좌표는 없습니다. 결측 좌표는 주소 부재 또는 좌표 변환 실패로 남은 행이며, AI 02의 격자 포인트 집계에는 들어가지 않습니다.

## 다음 단계 연결

AI 02는 이 단계의 land grid, 주유소 위치/속성 이력, 시설 포인트, 공시지가 격자를 결합해 날짜 x 500m 격자 패널을 만듭니다. AI 01은 가격 예측 자체보다 공간 feature의 원천 신뢰도를 결정하는 단계입니다.
