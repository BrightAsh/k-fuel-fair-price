# 07 Spatial Grid Build

전국 500m 격자를 생성하고 주유소, 시설, 섬 여부, 공시지가 등 공간 feature를 붙이는 단계입니다.

이 단계는 다음 작업에서 원본 노트북의 격자화 셀을 분리해 완성할 예정입니다. 현재는 폴더 구조와 산출물 공간만 준비했습니다.

## Inputs

- `06_external_data_collection` 외부 데이터 산출물
- 개별 주유소 일별 가격 및 메타데이터
- 시설 위치 데이터
- Natural Earth 기반 육지 경계
- 공시지가 데이터

## Outputs

- 500m 육지 격자
- 주유소 일별 격자 패널
- 시설 영향력 feature
- 지리 플래그 feature
- 공시지가 결합 패널
- 시각화 결과
