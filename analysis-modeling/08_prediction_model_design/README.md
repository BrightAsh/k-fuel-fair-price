# 08 Prediction Model Design

격자 단위 적정가격 예측 모델의 입력, 추론 구조, 검증 방식, 최종 산출물을 설계하는 단계입니다.

현재 프로젝트 방향은 웹 자동화에서 학습을 매일 반복하는 것이 아니라, 이미 생성된 모델을 가져와 추론만 수행하는 것입니다. 따라서 이 단계는 모델 학습 코드와 추론 배포용 산출물을 구분해 정리할 예정입니다.

## Inputs

- 격자 패널 데이터
- 전국 실제가격
- 정책 적용 적정가격
- 2단계/3단계 결과

## Outputs

- 휘발유 격자 예측 모델 bundle
- 경유 격자 예측 모델 bundle
- feature importance
- 검증 점수
- 2025~2026 예측 결과
- daily summary
- 최종 시각화 결과
