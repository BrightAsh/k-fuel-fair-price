# 03. Lag Analysis

이 단계는 국제 석유제품 가격 변동이 국내 유가에 어느 정도 시차를 두고 반영되는지 추정합니다. 원본 Colab 노트북의 `1단계: 시차 선정` 구간을 분리했고, 첨부된 `result.zip`의 실제 결과 파일을 바탕으로 결과까지 정리했습니다.

분석은 네 묶음으로 나뉩니다.

1. 국제 휘발유 제품가격 일별 변동 -> 주유소 휘발유 소비자가격 일별 변동
2. 국제 경유 제품가격 일별 변동 -> 주유소 경유 소비자가격 일별 변동
3. 국제 휘발유 제품가격 주간 변동 -> 정유사 휘발유 세전 공급가격 주간 변동
4. 국제 경유 제품가격 주간 변동 -> 정유사 경유 세전 공급가격 주간 변동

## 분석 방법

가격 수준값을 그대로 회귀하지 않고 로그 차분을 사용합니다.

```text
change = 100 * log(price).diff()
```

주요 처리 흐름은 다음과 같습니다.

1. 전처리 통합 데이터를 날짜순으로 정렬합니다.
2. 유종별 benchmark와 국내 target 컬럼을 선택합니다.
3. 주유소 분석은 일별 frame을 사용합니다.
4. 정유사 분석은 국제제품가격을 `W-SAT` 기준 주간 평균으로 만들고, 정유사 sparse 공급가격도 같은 주간 단위로 재구성합니다.
5. 정책 충격 구간을 제외합니다.
6. 국내 가격 변화의 자기시차 `p`와 국제 가격 변화의 시차 `q`를 탐색합니다.
7. 후보 모델은 안정성, Ljung-Box 검정, 국제가격 block 검정, rolling OOS 성능을 기준으로 평가합니다.
8. 최종 모델에서 impulse response path를 계산합니다.
9. IRF의 peak lag, half-mass lag, mean lag를 기준으로 대표 반영 시차를 산정합니다.

주유소 일별 분석은 `max_p=60`, `max_q=60`, `response_h=90`을 사용합니다. 정유사 주간 분석은 주간 단위 분석이며 결과의 lag period는 주 단위로 해석해야 합니다.

## IRF 기반 시차 해석

| 분석 | peak lag | half-mass lag | mean lag | 누적 반응 | 대표 시차 |
| --- | ---: | ---: | ---: | ---: | ---: |
| 휘발유 주유소 | 8일 | 16일 | 20.27일 | 0.216 | 20일 |
| 경유 주유소 | 8일 | 21일 | 23.05일 | 0.326 | 23일 |
| 휘발유 정유사 | 1주 | 3주 | 8.12주 | 0.755 | 8주 |
| 경유 정유사 | 1주 | 1주 | 6.29주 | 0.816 | 6주 |

주유소 소비자가격은 국제제품가격 충격의 초기 peak가 8일 근처에서 나타나지만, 누적 반응의 평균 시점은 휘발유 약 20일, 경유 약 23일입니다. 따라서 이후 적정가격 모델에서 전국 주유소 소비자가격을 설명할 때는 휘발유 20일, 경유 23일 정도의 대표 시차를 사용하는 것이 적절합니다.

정유사 주간 분석은 lag 단위가 주입니다. `recommended_shift_days`라는 기존 컬럼명이 일부 summary에 남아 있더라도, 정유사 주간 분석에서는 이를 일 단위로 해석하면 안 됩니다. 휘발유 정유사는 약 8주, 경유 정유사는 약 6주이며, 각각 약 56일과 42일에 해당합니다.

## 결과 해석

휘발유 주유소 소비자가격은 국제 휘발유 92RON 제품가격 변동을 약 20일 후 평균적으로 반영하는 것으로 정리됩니다. peak lag는 8일이지만, 누적 반응의 중심이 20일 부근에 있어 단일 대표 시차로는 20일이 더 적절합니다.

경유 주유소 소비자가격은 국제 경유 0.001% 제품가격 변동을 약 23일 후 평균적으로 반영합니다. 휘발유보다 대표 시차가 조금 더 길고, 선택된 국제가격 lag 수 `q=34`도 더 큽니다.

정유사 세전 공급가격은 주간 분석 기준으로 국제제품가격 충격에 빠르게 peak 반응을 보이지만, 누적 반응의 중심은 휘발유 8주, 경유 6주로 계산됩니다. 이 결과는 정유사 공급가격 layer의 진단 및 정책 검토에는 유용하지만, 주유소 소비자가격 적정가격 산정의 직접 anchor로는 주유소 일별 시차 결과가 더 적합합니다.
