# 03. Lag Analysis

## 목적

`03_lag_analysis.ipynb`는 국제 제품가격 변화가 국내 가격에 반영되는 시차를 추정합니다. 02 단계에서 선택된 제품 benchmark를 사용해 두 층위의 시차를 따로 계산합니다.

1. 국제 제품가격 일별 변화 -> 주유소 소비자가격 일별 변화
2. 국제 제품가격 주간 변화 -> 정유사 세전 공급가격 주간 변화

노트북 구조는 markdown 5개, code 24개, 총 2,328줄입니다. 출력이 저장된 code 셀은 8개이며, 네 개 분석의 진행 상황과 최종 표가 포함되어 있습니다.

일별 분석 기간은 `2008-05-10 ~ 2025-12-31`입니다. 정유사 주간 시계열은 첫 실제 주간 종료일을 `2008-05-10`으로 두고 `W-SAT` 주기로 재구성합니다.

## 공통 모델 로직

가격 수준 자체가 아니라 로그 차분 변화율을 모델링합니다.

```text
change = 100 * log(price).diff()
```

각 분석은 국내 가격의 자기회귀 lag `p`와 국제 가격 lag `q`를 grid search합니다. 회귀는 OLS를 사용하고, 최종 요약은 HAC 공분산으로 출력합니다. 진단에는 ADF, KPSS, Ljung-Box, 국제 가격 lag block 유의성, AR 안정성, out-of-sample 예측 오차가 포함됩니다.

정책 또는 외생 충격 기간은 정상 반영 시차 추정에서 제외합니다. 주요 제외 구간은 2008년 유류세 인하, 2011년 정유사 100원 인하, 2018~2019년 유류세 인하, 2021년 이후 유류세 인하/환원 구간, 2026년 가격상한제 구간입니다.

## 분석별 해석

### 휘발유 주유소 일별

휘발유 92RON 원화/리터 변화가 보통휘발유 전국 평균 소비자가격에 반영되는 소비자 체감 시차를 추정했습니다. 선택 모델은 `p=9`, `q=14`입니다. IRF peak는 8일, half-mass는 16일, 평균 lag는 약 20.3일입니다. 따라서 휘발유 주유소 가격은 국제제품가 충격을 약 3주에 걸쳐 흡수하는 구조로 해석합니다.

### 경유 주유소 일별

경유 0.001 원화/리터 변화가 자동차용경유 전국 평균 소비자가격에 반영되는 시차를 추정했습니다. 선택 모델은 `p=21`, `q=34`입니다. peak lag는 휘발유와 같은 8일이지만 half-mass는 21일, 평균 lag는 약 23.0일로 더 깁니다. 경유 소비자가격은 휘발유보다 조금 더 긴 누적 반영 시차를 보입니다.

### 휘발유 정유사 주간

국제 휘발유 제품가격 주간 변화가 정유사 세전 휘발유 공급가격에 반영되는 시차를 추정했습니다. 선택 모델은 `p=13`, `q=12`입니다. peak는 1주차에 매우 빠르게 나타나지만 평균 lag는 8.1주입니다. 누적 반응은 0.755로, 04 단계에서 정유사 주간 진단 모델의 IRF feature로 사용됩니다.

### 경유 정유사 주간

국제 경유 0.001 제품가격 주간 변화가 정유사 세전 경유 공급가격에 반영되는 시차를 추정했습니다. 선택 모델은 `p=13`, `q=13`입니다. peak와 half-mass는 모두 1주로 빠르지만, 평균 lag는 6.3주입니다. 누적 반응은 0.816입니다.
