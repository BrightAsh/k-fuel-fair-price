# 05. Policy Application

## 목적

`05_policy_application.ipynb`는 04 단계에서 만든 정책 미반영 적정가격대에 국내 유류세 인하 정책을 반영합니다. 정책 적용 후 적정가격, 적정 범위, 판정을 다시 계산하고, 정유사 최고가격제는 정유사 주간 진단 모델 기준으로 별도 점검합니다.

노트북 구조는 markdown 4개, code 5개, 총 1,726줄입니다. 출력이 저장된 code 셀은 4개이며, 데이터 기간, 입력 파일 탐색 결과, CSV 저장 로그, 연도별 그래프 생성 로그가 포함되어 있습니다.

## 입력

| 입력 | 사용 내용 |
|---|---|
| 04 `gasoline_production_predictions_full_calendar.csv` | 휘발유 정책 미반영 적정가격대 |
| 04 `diesel_production_predictions_full_calendar.csv` | 경유 정책 미반영 적정가격대 |
| 04 `gasoline_refinery_weekly_predictions.csv` | 휘발유 정유사 세전 적정가격 진단 |
| 04 `diesel_refinery_weekly_predictions.csv` | 경유 정유사 세전 적정가격 진단 |
| 정책 파일 `data-analysis/00_data_collection/outputs/z_pa_policy/korea_fuel_tax_price_policies.csv` | 유류세 인하율, 최고가격제 상한, 정책 기간 |
| 01 통합 데이터 | 국내 실제가격, 유류세 항목, 정유사 실제 세전 공급가격 |

노트북 출력 기준 데이터 기간은 `2008-04-15 ~ 2026-06-09`입니다.

## 정책 분류와 처리 방식

정책 적용 가능 여부 요약은 28행입니다.

| 구분 | 건수 | 처리 |
|---|---:|---|
| 카테고리 0, 유류세 인하 | 24 | 주유소 적정가격과 band를 수치 shift |
| 카테고리 1, 정유사 최고가격제 | 4 | 정유사 단계 감사로 분리 |

`정책_적용가능여부_요약.csv`의 처리결과는 `수치반영` 24건, `정유사단계감사` 4건입니다. 유류세 인하는 주유소 가격대에 직접 반영하지만, 최고가격제는 소비자가격 band에 직접 더하지 않고 정유사 주간 가격의 상한 초과 여부를 점검합니다.

## 유류세 인하 반영 로직

유류세 인하율은 기준 유류세에 곱해 감면액을 계산합니다. 실제 소비자가격 관점에서는 VAT 포함 참고 감면액을 정책 shift로 사용합니다.

```text
정책감면액_exVAT = 기준유류세_exVAT * 인하율
정책적용_시프트 = 정책감면액_exVAT * 1.1
적정가격_정책적용 = 적정가격_미정책 - 정책적용_시프트
적정범위_정책적용 = 적정범위_미정책 - 정책적용_시프트
```

2008년 유류세 인하 기간은 감면액 자체는 계산되지만, 04 적정가격대가 아직 충분히 생성되지 않은 초반 구간이라 평균 정책효과가 비어 있습니다.

## 최종 해석

정책 미반영 기준으로는 2021년 이후 유류세 인하 기간의 국내 가격이 대부분 적정가격대 아래로 보입니다. 그러나 유류세 인하분을 적정가격대에도 반영하면 휘발유는 정책 기간 적정일이 4일에서 615일로, 경유는 9일에서 192일로 늘어납니다. 이는 정책으로 낮아진 가격을 단순히 과소가격으로 해석하면 안 된다는 점을 보여줍니다.

정유사 최고가격제는 실제 세후 추정 정유사 가격 기준으로 상한 초과가 없었습니다. 다만 모델 적정가격 기준으로는 일부 주간에서 상한보다 높은 적정 세후 가격이 추정되어, 최고가격제가 정유사 단계에서 가격을 누르는 정책 제약으로 작동했을 가능성을 별도로 점검할 수 있습니다.
