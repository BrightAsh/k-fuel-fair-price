# Manual Inputs

이 폴더는 웹 대시보드가 필요한 공개용 보조 데이터를 임시로 받는 공간입니다. 핵심 분석이나 AI 학습을 대체하는 자료가 아니라, 자동 산출물이 아직 준비되지 않았거나 추가 확인이 필요한 항목을 화면에 연결하기 위한 보강 입력입니다.

## 입력 성격

| 입력 | 화면에서 쓰는 의미 |
|---|---|
| 지역별 오늘 요약 | 지역 카드와 지도에 표시할 실제 가격, 적정가격, 판정 |
| 주유소 검색 인덱스 | 사용자가 주변 주유소를 검색할 때 보여줄 공개 정보 |
| 가격 추세 보강 | 차트에서 비어 있는 기간을 임시로 채우는 기간 데이터 |
| 학습 데이터 coverage | AI 학습에 사용된 격자/주유소/시설/공시지가 분포 요약 |

## 데이터 원칙

| 원칙 | 설명 |
|---|---|
| 공개 가능한 값만 사용 | 개인정보나 내부 원천 컬럼은 포함하지 않습니다. |
| 날짜와 유종 명확화 | 가격 데이터는 기준일과 휘발유/경유 구분이 분명해야 합니다. |
| 산출물과 같은 의미 유지 | 수동 입력도 data-analysis와 ai-model의 산출물 의미를 따라야 합니다. |
| 임시 입력으로 취급 | 최종 기준은 자동 산출물이며, 수동 입력은 보강용입니다. |

## 다음 단계 연결

수동 입력은 웹 표시용 데이터로 변환된 뒤 대시보드에 반영됩니다. 장기적으로는 data-analysis와 ai-model 산출물이 충분히 안정되면 수동 보강 비중을 줄이고 자동 갱신 결과를 우선 사용합니다.
